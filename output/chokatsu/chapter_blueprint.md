# 章別設計書: 初心者のための腸活入門

## 全体設計

### 大アーク（感情遷移）
はじめに: 共感＋約束 → 第1章: 問題認識 → 第2章: 危機感 → 第3章: 希望 → 第4章: 自信 → 第5章: 自立 → おわりに: 行動開始

### 追体験ペルソナ
- ペルソナA: 佐藤美咲（34歳・事務職） — 便秘、夕方の張り、肌荒れに悩み、第1章で迷いを提示し、第4章で買い物リスト固定によって改善の手応えを得る。
- ペルソナB: 山下健太（41歳・営業職） — 外食と出張が多く、下痢とガスを繰り返し、第2章から第5章で自分のルールを作る。
- ペルソナC: 田中莉子（29歳・美容職） — SNSの腸活情報を試しすぎてお腹が張り、第3章で2週間プログラムを実践する。
- ペルソナD: 中村陽子（45歳・小学生の母） — 子どもの便秘と家族の食事作りに悩み、第4章で家族対応の工夫を学ぶ。

- 本書の約束: スーパーで買える食品だけで2週間の型を作る — Layer 2 / Layer 3
- 医療との線引き: 強い症状は受診を優先する — 厚生労働省情報

### 一次情報・ストーリー配分

- パターンB: 大正健康ナビ、日清製粉、厚生労働省情報を読者向けに再構成。

- 1.4 セルフケアで済ませてはいけない症状: 下痢、血便、強い痛み、体重減少 — 厚生労働省

### 一次情報・ストーリー配分
- パターンA: 佐藤美咲が毎朝ヨーグルトを食べても夕方に張る話。
- パターンB: 外科医石黒ドクター系の医師解説と厚生労働省情報を安全性の話に変換。
- パターンC: 便の回数、形、張り、食事、睡眠を記録するワーク。

### この章では扱わない内容

- 2.4 サプリや整腸剤の前に整えるべき土台: 食事、水分、受診判断 — Yahoo知恵袋 / Amazonレビュー傾向
