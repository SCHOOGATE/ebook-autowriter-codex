# 章別設計書: 忙しい人のための時間管理入門

作成日: 2026-06-09

## 第1章: 忙しさの正体を見える化する
- 読者の悩み: 予定表は埋まっているのに、何に時間を使ったのか説明できない状態
- 章の約束: 時間が足りないという感覚を、調整できる事実へ変える
- 対応する本文方針: 共感シーン、原因、即実践アクション、次章への橋渡しを必ず入れる
- 1.1 忙しい日の記憶はなぜ曖昧になるのか: 朝から会議、昼休みの返信、夕方の急な依頼で一日が消える会社員 / 時間の記憶を十五分単位で戻す / その日の最後に三つの時間帯だけを書き出す
- 1.2 予定と実績を分けて見る: 資格学習を予定していたのに、実際には家事とメールで夜が終わる人 / 予定表と実績メモを同じものとして扱わない / 予定の横に実際に使った時間を短く記録する
- 1.3 急ぎの用事が重要な目標を押し出す仕組み: 返信が早い人ほど、長期の学習や副業準備が後回しになる職場 / 緊急度と重要度を同じ棚に置かない / 急ぎの作業に上限時間をつける
- 1.4 余白がない計画はなぜ一日で崩れるのか: 朝の段階で予定を詰め込み、午後の遅れで全部を諦める人 / 余白を怠けではなく調整費として扱う / 一日の二割を空白として先に残す
- 1.5 時間を奪う習慣を静かに減らす: 寝る前の短い動画が続き、翌朝の集中力が落ちる生活 / 悪い習慣を根性で止めず、入口を遠ざける / スマホを充電する場所を寝室の外へ移す

## 第2章: 目標を行動へ翻訳する
- 読者の悩み: 目標は書けるのに、今日何をすればよいか決められない状態
- 章の約束: 大きな目標を、迷わず始められる小さな行動へ変える
- 対応する本文方針: 共感シーン、原因、即実践アクション、次章への橋渡しを必ず入れる
- 2.1 願望と目標と行動を分ける: 英語を話せるようになりたいと思いながら教材だけが増えていく人 / 曖昧な願いを測れる目標に直す / できるようになりたい場面を一つだけ決める
- 2.2 数字は自分を責めるためではなく迷いを減らすために使う: 毎日勉強すると決めても、何分やれば合格なのかわからない人 / 数字を罰ではなく判断基準として使う / 週合計の学習時間を先に決める
- 2.3 月の目標を今週の予定に落とす: 副業を始めたいが、週末になると疲れて手が止まる人 / 月単位の希望を週単位の成果物に変える / 今週の完成物を一つだけ決める
- 2.4 障害を先に書くと継続率が上がる: 帰宅後に作業するつもりでも、夕食後に眠くなる人 / 失敗する場面を事前に予定へ組み込む / 疲れた日の代替行動を用意する
- 2.5 最初の一歩を小さくすると再開しやすい: 完璧に準備してから始めたいと思い、何週間も着手できない人 / 開始の負荷を限界まで下げる / 五分で終わる作業名に書き換える

## 第3章: 一週間を設計する
- 読者の悩み: 毎朝予定を考えて疲れ、結局いつもの緊急対応に流される状態
- 章の約束: 週のはじめに重要な時間を確保し、日々の判断を軽くする
- 対応する本文方針: 共感シーン、原因、即実践アクション、次章への橋渡しを必ず入れる
- 3.1 週のはじめに予定を全部眺める: 月曜朝にメールを開いた瞬間、週の主導権を失う人 / 先に全体像を見てから細部へ入る / 固定予定、締め切り、自分の目標時間を色分けする
- 3.2 今日の三つの優先を決める: ToDoを十個書いて、どれも中途半端に残る人 / 一日の勝ち筋を三つに絞る / 朝に今日終われば安心できる三件だけを書く
- 3.3 集中枠は長さより開始時刻が大切です: 時間が空いたらやると考えて、結局一度も空かない人 / 集中を気分ではなく時刻に結びつける / 三十分の集中枠を予定表に先置きする
- 3.4 連絡と確認の時間をまとめる: 通知が鳴るたびに作業を止め、夕方に疲れ切る人 / 反応の速さと成果を切り分ける / 返信時間を午前と午後に二回だけ作る
- 3.5 日曜夜の短い見直しで翌週が軽くなる: 週末に休んだはずなのに月曜が怖くなる人 / 翌週の不安を紙に出して小さくする / 日曜夜に十五分だけ予定と荷物を整える

## 第4章: 先延ばしから戻る
- 読者の悩み: 一度計画が崩れると、自分を責めて再開まで時間がかかる状態
- 章の約束: 失敗を終わりにせず、次の行動へ戻す手順を持つ
- 対応する本文方針: 共感シーン、原因、即実践アクション、次章への橋渡しを必ず入れる
- 4.1 先延ばしは性格ではなく摩擦です: 資料作成を始める前に机を片づけ始めてしまう人 / 行動の前にある小さな摩擦を見つける / 始める前に必要なものを一つにまとめる
- 4.2 五分だけ始めると脳の抵抗が下がる: 二時間必要だと思って手をつけられない副業作業 / 完成ではなく着手を目的にする / 五分だけファイルを開いて見出しを書く
- 4.3 朝昼夜の小さな合図を作る: 毎日違う時間に動こうとして習慣が定着しない人 / 行動を既存の生活動作に結びつける / 朝の飲み物、昼の席戻り、夜の充電を合図にする
- 4.4 予定が崩れた日は復旧だけを目標にする: 残業で夜の学習を失い、週全体を諦めそうになる人 / 遅れを取り返すより復旧を優先する / 翌日の予定から一つだけ削って学習枠を戻す
- 4.5 記録は反省文ではなく次の設計図です: できなかった理由を書くだけで気持ちが沈む人 / 記録を責任追及ではなく条件調整に使う / できた条件とできなかった条件を一行ずつ残す

## 第5章: 仕事・学習・副業を同時に回す
- 読者の悩み: どの役割も大切で、優先順位をつけるたびに罪悪感が出る状態
- 章の約束: 複数の役割を競わせず、時間配分として現実的に整える
- 対応する本文方針: 共感シーン、原因、即実践アクション、次章への橋渡しを必ず入れる
- 5.1 会社の目標管理は日常業務から作る: 事務職で新しい目標が思いつかず、面談前に困る人 / いつもの仕事を改善行動として言語化する / 処理時間、ミス防止、引き継ぎの三方向から目標を書く
- 5.2 学習計画は教材ではなく復習日から決める: 新しい教材を買って満足し、記憶が残らない人 / 勉強時間より思い出す機会を設計する / 学んだ翌日と三日後に短い復習を置く
- 5.3 副業はまとまった時間より固定された小枠で進む: 週末だけ頑張ろうとして平日の疲れに負ける人 / 副業を大作業ではなく工程に分ける / 調査、下書き、公開準備を別の日に置く
- 5.4 家事と生活タスクを敵にしない: 生活を整える時間を削って作業し、翌週に反動が出る人 / 生活維持を目標達成の土台として扱う / 買い物、洗濯、睡眠を先に予定へ入れる
- 5.5 月末に目標を更新する: 一カ月前の目標が今の状況に合わなくなっている人 / 目標を守るより更新する勇気を持つ / 続ける、減らす、やめるの三択で見直す
