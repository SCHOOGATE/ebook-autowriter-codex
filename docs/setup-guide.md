## 1. はじめに読んでください

### このツールでできること

「{テーマ}の電子書籍を制作」と入力するだけで、AIが自動的に電子書籍を1冊まるごと作ってくれます。

作ってくれるもの：
- 本文（25,000字以上の原稿）
- 表紙画像
- Amazon出品用のデータ（タイトル案、キーワード、紹介文など）
- 商品ページ用の画像4枚

### 必要なもの

- パソコン（Windows）
- インターネット接続
- クレジットカード（ChatGPTの月額料金の支払い用）

### かかる費用

- ChatGPT Plus プラン：月額 $20（約3,000円）。GPT-5.5はPlusに含まれています
- APIクレジットやクレジットカードの追加登録は一切不要です
- このガイドでインストールするソフトは全て無料です

### セットアップにかかる時間

初回のセットアップは約30分です。
一度設定すれば、あとは「{テーマ}の電子書籍を制作」と入力するだけで何冊でも作れます。

## 2. Node.js のインストール

Node.js（ノードジェイエス）とは：このあとインストールする「Codex CLI」というツールを動かすために必要なソフトです。裏方として動くだけなので、普段意識する必要はありません。

### やること

1. ブラウザで https://nodejs.org/ を開きます

2. ページの中央あたりに、緑色の大きなボタンが2つ並んでいます。左側の「LTS（推奨版）」と書かれたボタンをクリックします

3. ダウンロードが完了したら、そのファイルをダブルクリックして開きます

4. インストール画面が表示されます。以下の順にボタンをクリックしていきます：
   - 「Next」をクリック
   - ライセンス同意画面 → チェックを入れて「Next」をクリック
   - インストール先の画面 → そのまま「Next」をクリック
   - カスタムセットアップ画面 → そのまま「Next」をクリック
   - 「Install」をクリック
   - 「このアプリがデバイスに変更を加えることを許可しますか？」→「はい」をクリック

5. インストールが進みます。完了画面が出たら「Finish」をクリック

### 確認方法

1. キーボードの左下にある「Windows キー」（窓のマークのキー）を押します
2. そのまま `cmd` と入力します
3. 検索結果に「コマンドプロンプト」と表示されるので、クリックして開きます
4. 黒い画面に以下の文字を入力して、Enterキーを押します：
   ```
   node --version
   ```
5. 「v20.xx.x」のように「v」から始まる数字が表示されれば成功です

うまくいかない場合：パソコンを一度再起動してから、もう一度試してください。

## 3. Git のインストール

Git（ギット）とは：書籍制作に必要なファイルをインターネットからダウンロードするために使うソフトです。こちらも裏方として動くだけです。

### やること

1. ブラウザで https://git-scm.com/ を開きます
2. 画面右側にある「Download for Windows」というボタンをクリックします
3. ダウンロードページが表示されます。「Standalone Installer」の「64-bit Git for Windows Setup」をクリックします
4. ダウンロードが完了したら、ファイルをダブルクリックして開きます
5. インストール画面が表示されます。設定は一切変更せず、以下の操作だけ繰り返します：
   - 「Next」をクリック（何度か繰り返します。全部で約10画面あります）
   - 最後に「Install」をクリック
   - 「このアプリがデバイスに変更を加えることを許可しますか？」→「はい」をクリック
6. インストールが進みます。完了画面が出たら「Finish」をクリック

### 確認方法

コマンドプロンプトで以下を入力してEnterキーを押します：
```
git --version
```
「git version 2.xx.x.windows.x」のように表示されれば成功です。

## 4. Python のインストール

Python（パイソン）とは：書籍の品質検証と画像のリサイズ処理に使うソフトです。

### やること

1. ブラウザで https://www.python.org/downloads/ を開きます
2. ページ上部にある黄色い「Download Python 3.xx.x」ボタンをクリックします
3. ダウンロードが完了したら、ファイルをダブルクリックして開きます
4. **ここが最も重要な手順です。** インストール画面の一番下に、以下のチェックボックスがあります：
   - 「Add python.exe to PATH」
   - **必ずこれにチェックを入れてください**
5. チェックを入れたら、「Install Now」をクリックします
6. 「このアプリがデバイスに変更を加えることを許可しますか？」→「はい」をクリック
7. インストールが進みます。完了画面が出たら「Close」をクリック

### 確認方法

コマンドプロンプトを**一度閉じて、新しく開き直してください**（開き直さないとPythonが認識されません）。

```
python --version
```
「Python 3.xx.x」のように表示されれば成功です。

うまくいかない場合：手順4で「Add python.exe to PATH」にチェックを入れ忘れた可能性があります。一度Pythonをアンインストール（「設定」→「アプリ」→「Python」→「アンインストール」）してから、手順1からやり直してください。

## 5. Codex CLI のインストール

Codex CLI（コーデックス シーエルアイ）とは：「{テーマ}の電子書籍を制作」と入力するだけでAIが書籍を自動制作してくれる、このツールの本体です。

### やること

1. コマンドプロンプトを開きます（Windowsキー → cmd → コマンドプロンプト）
2. 以下を入力してEnterキーを押します：
   ```
   npm install -g @openai/codex
   ```
3. 画面に文字がたくさん流れます。これはインストールが進んでいる様子です。数分かかることがあるので、そのまま待ってください。途中で画面を閉じないでください。
4. 最後に「added xx packages」のような表示が出たら完了です

### 確認方法

```
codex --version
```
「0.xxx.0」のような数字が表示されれば成功です。

うまくいかない場合：「'codex' は、内部コマンドまたは外部コマンド...として認識されていません」と出る場合は、コマンドプロンプトを閉じて開き直してから再試行してください。それでもダメな場合はパソコンを再起動してください。

## 6. Codex CLI にログイン

ChatGPTのアカウントを使ってログインします。APIキーの発行やクレジットカードの追加登録は一切不要です。

### やること

1. コマンドプロンプトで以下を入力してEnterキーを押します：
   ```
   codex login
   ```
2. ブラウザが自動的に開きます。ChatGPTのログイン画面が表示されます
3. いつもChatGPTにログインしている方法でログインします（メールアドレス＋パスワード、またはGoogleログイン）
4. ログインに成功すると、ブラウザに「Signed in to Codex」または「You can close this window」のようなメッセージが表示されます。ブラウザのこのタブは閉じてOKです
5. コマンドプロンプトに戻ると、「Successfully logged in」と表示されています

うまくいかない場合：
- ブラウザが自動で開かない → コマンドプロンプトに表示されたURL（https://...）をコピーして、手動でブラウザのアドレスバーに貼り付けてください
- ChatGPTにログインできない → ChatGPT PlusまたはProの有効なアカウントが必要です

## 7. Chrome をデバッグモードで起動する設定

Codex CLIがChromeブラウザを自動操作して、GPT-5.5の画像生成機能を使い、表紙画像や本文画像を自動で作成します。そのために、特別な起動方法が必要です。

### やること

コマンドプロンプトで以下をそのままコピーして貼り付け、Enterキーを押します：
```
powershell -ExecutionPolicy Bypass -File scripts\create_shortcuts.ps1
```
※ 上の枠内は1行です。途中で改行せず、全体をそのままコピーして貼り付けてください。

デスクトップに2つのアイコンが自動で作成されます：
- **Chrome book creation** — 書籍制作前にこのChromeを起動する（画像自動生成に必要）
- **BookRestore** — 制作完了後にダブルクリックで完成データのフォルダを開く

### 動作確認

1. デスクトップの「Chrome book creation」をダブルクリックして起動します
2. Chromeが開き、ChatGPTのページが自動で表示されます
3. ChatGPTにログインしてください（初回のみ必要。2回目以降は自動ログインされます）

注意事項：
- 書籍制作の前には、必ず「Chrome book creation」からChromeを起動してください
- 普段使いのChromeを閉じる必要はありません。別プロファイルで動くため同時に起動できます

## 8. 書籍制作リポジトリのダウンロード

書籍制作に必要なファイル一式をダウンロードします。

### やること

1. コマンドプロンプトを開きます（Windowsキー → cmd → コマンドプロンプト）
2. まず、ファイルを保存するフォルダを作ります。以下を入力してEnter：
   ```
   mkdir %USERPROFILE%\repos
   ```
   （「既に存在します」というエラーが出ても問題ありません。そのまま次に進んでください）
3. 次に、そのフォルダに移動します：
   ```
   cd %USERPROFILE%\repos
   ```
4. ファイルをダウンロードします：
   ```
   git clone https://github.com/SCHOOGATE/ebook-autowriter-codex.git
   ```
5. 画面に文字が流れ、「done.」のような表示が出たら完了です
6. ダウンロードしたフォルダに移動します：
   ```
   cd ebook-autowriter-codex
   ```

### 確認方法

```
dir
```
ファイルやフォルダの一覧が表示されれば成功です。

## 9. Chrome DevTools MCP の追加

Chrome DevTools MCP とは：Codex CLI が Chrome ブラウザを操作するための「橋渡し」をする仕組みです。この設定をすることで、画像の自動生成が可能になります。

### やること

コマンドプロンプトで以下を入力してEnter：
```
codex mcp add chrome-devtools -- npx -y chrome-devtools-mcp@latest
```
※ 上の枠内は1行です。途中で改行せず、全体をそのままコピーして貼り付けてください。

「Added global MCP server 'chrome-devtools'.」と表示されれば成功です。

うまくいかない場合：「codex が認識されない」と出る場合は、Step 5のCodex CLIインストールをもう一度実行してください。

**セットアップ完了です。次の「電子書籍を作ってみる」に進んでください。**

## 10. 電子書籍を作ってみる

### 10-1. Chrome book creation を起動する

1. デスクトップにある「Chrome book creation」をダブルクリックします
2. Chromeが開いたら、ChatGPTにログイン済みか確認します
3. ChatGPTにログインできたら、Chromeはそのまま開いておいてください（閉じないでください）

### 10-2. コマンドプロンプトで制作を開始する

1. Windowsキーを押す → `cmd` と入力 → 「コマンドプロンプト」をクリック
2. 以下をそのままコピーして貼り付け、Enterキーを押します：
   ```
   cd %USERPROFILE%\repos\ebook-autowriter-codex
   ```
3. 続けて、以下を貼り付けてEnter（最新版に更新されます）：
   ```
   git pull
   ```
4. 続けて、以下をそのままコピーして貼り付け、Enterキーを押します：
   ```
   codex --model gpt-5.5 --dangerously-bypass-approvals-and-sandbox "AGENTS.mdの指示に従って「{テーマ}の電子書籍を制作」を開始してください。"
   ```
   ※ `{テーマ}` の部分を作りたい本のテーマに置き換えてください（例：「AI副業」「猫のアロマケア」）
   ※ 上の枠内は1行です。途中で改行せず、全体をそのままコピーして貼り付けてください

5. 「Reading additional input from stdin...」と表示されたら、準備完了です。AIが動き始めます

コピーの仕方：上の文字を左クリックしながらなぞって青く選択 → 右クリック →「コピー」。コマンドプロンプトに貼り付けるときは、黒い画面の上で右クリックすると貼り付けられます。

### 10-3. AIの質問に答える

AIが順番に質問してきます。コマンドプロンプトの黒い画面に直接文字を入力して、Enterキーを押して答えます。

**質問1: 著者名と参考資料**
```
テーマ「○○」で電子書籍の制作を開始します。以下を教えてください。
1. 著者名（必須）
2. 参考資料（任意）
```
→ 例：`著者名: 田中太郎` と入力してEnter
※ 参考資料は入力しなくてもOKです。テーマと著者名だけで制作できます。

**質問2: タイトルの選択**
AIが10個のタイトル案を提案してきます。気に入った番号を入力してEnterを押します。
→ 例：`8番でOK`

**質問3: 目次（構成）の確認**
AIが5章構成の目次案を提案してきます。
→ そのままでよければ `OK` と入力してEnter

### 10-4. 自動制作を見守る

目次を承認すると、AIが以下をすべて自動で行います：
- 原稿執筆（25,000字）
- 出版メタデータ生成
- Kindle申請データ生成
- 表紙画像生成（Chromeを自動操作してGPT-5.5の画像生成機能で生成）
- A+コンテンツ画像4枚生成

途中でAIが確認を求めてきたら「OK」と入力してEnterを押してください。

すべての工程が終わると「電子書籍の制作が完了しました」と表示されます。

### 10-5. 完成データを取得する

1. デスクトップにある「BookRestore」アイコンをダブルクリックします
2. 完成データが入ったフォルダが自動で開きます

以下のファイルが入っています：
- `manuscript.md` — 原稿（25,000字以上）
- `manuscript.html` — HTML原稿（Word変換用）
- `listing.txt` — Amazon出品用データ
- `kindle_application.txt` — Kindle申請データ
- `images/cover.jpg` — 表紙画像
- `images/aplus_1〜4.png` — 商品ページ用画像4枚

### 2冊目以降の制作

Step 10-1〜10-5をもう一度繰り返すだけです。毎回新しいテーマで制作できます。
前に作った書籍のデータが消えることはありません。テーマごとに別のフォルダに保存されます。

## 11. よくある質問

**Q: 制作中にエラーが出ました**
A: コマンドプロンプト（黒い画面）を右上の「×」で閉じて、10-2からやり直してください。

**Q: 画像の品質が悪いです**
A: ChatGPTにログインした状態でChromeが開いているか確認してください。ログインしていないと画像が低品質になることがあります。

**Q: 途中で止まってしまいました**
A: 5分以上何も表示されない場合は、キーボードの Ctrl キーと C キーを同時に押してキャンセルしてください。その後、10-2からやり直してください。

**Q: 表紙のデザインを変えたいです**
A: 表紙プロンプトの確認のときに、「もっとシンプルにしてください」「色を青系にしてください」のように修正の指示を入力できます。

**Q: 原稿の文章を修正したいです**
A: 完成フォルダの中にある manuscript.md をメモ帳やWordで開いて、直接編集できます。

**Q: 2冊目を作ったら前のデータは上書きされますか？**
A: 上書きされません。テーマごとに別のフォルダ（output/テーマ名/）に保存されるので、前のデータはそのまま残ります。

**Q: Chrome book creation を起動し忘れて制作を始めてしまいました**
A: 表紙画像の生成段階でエラーになります。Ctrl+C で中断し、Chrome book creation を起動してから10-2のコマンドで再開してください。

**Q: ツールを最新版にアップデートするには？**
A: 制作開始時に `git pull` を実行すれば自動で最新版に更新されます（10-2の手順3に含まれています）。

## 困ったときは

AI出版ラボのコミュニティで質問してください。
コマンドプロンプトに表示されたエラーメッセージのスクリーンショット（画面の写真）を添えて「ここで止まっています」と伝えていただくと、すぐに解決できます。
